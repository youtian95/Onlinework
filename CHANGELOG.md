# Changelog

## [0.0.5] - 2026-03-08

### 修改

- [x] 题目页面新增独立 PDF 上传位置，学生可单独上传 PDF，且不与答案提交强绑定。
- [x] Git不追踪题目文件夹下的题目，除了demo开头的题目文件夹。
- [x] Bug：当题目脚本在单字段提交或异常输入下抛错时，不让整个接口 500，而是将本次提交字段判为错误。
- [x] 提交错误答案时振动提示。

## [0.0.4] - 2026-03-03

### 修改

- [x] 默认每个空格分数为 1 分。
- [x] 默认每个空格尝试次数为 3 次。
- [x] 题目md文档中 meta.inputs 允许不填写，默认为空对象 {}。
- [x] 如果题目md文档中有多个相同的输入框名称，系统应该能够正确处理，只需要填对其中一个输入框，其他相同输入框就会同步。并且计算分数时应该考虑所有相同名称的输入框，相同名称的输入框分数不叠加。

## [0.0.3] - 2026-02-08

### 修改

- [x] 添加游客模式，允许用户在不注册账户的情况下浏览部分内容。
- [x] 添加图标。
- [x] 文档添加部署说明。
- [x] 增加Umami统计代码，监控用户访问数据。
- [x] 增加Github链接，方便用户访问项目源代码和提交问题。

## [0.0.2] - 2026-02-06

### 修改

- [x] 添加说明文档页面，在封面、学生端的顶部导航栏中增加说明文档链接，指向该页面。
- [x] 添加页面的底部信息。
- [x] 前端路由与 Nginx 配置。
- [x] 修改 `docker-compose.yml` 端口映射为 `5050:80`，避免与服务器面板（如 1Panel）的默认 80 端口冲突。
