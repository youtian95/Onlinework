## [0.1.0] - 2026-03-17

### 修改

- [x] 重复题目ID上传时，系统进行提示逻辑是覆盖更新。
- [x] 题目“立即截止”的按钮，对于已经截止的题目，允许重新开放。
- [x] 大更新：团队作业功能。
  - 教师端：
    - 对于某一个团队作业，老师可以设置队伍的人数和队伍数量（在meta信息中），一个团队作业下面有跟队伍人数一致的子题目数量。
  - 学生端：
    - 学生进入团队作业页面之后，如果没有加入队伍，那么就先必须选择一个队伍，然后才进入题目界面。题目界面增加一个按钮可以查看当前全部队伍的学生列表。
    - 可以重新选择队伍，但是会清空之前的提交记录，给学生提示是否要重新选择队伍。
    - 加入某一个队伍之后，题目界面学生可以任意认领一个子题目。学生只能修改自己子题目的答案，但是队伍内所有学生都能看到其他子题目提交结果。队伍内也可以切换选择其他未被选择的题目，但是切换题目会清空之前的提交记录，给学生提示是否要切换题目。
    - 子题目显示可以进行折叠，默认展开当前学生认领的子题目，其他子题目默认折叠。
    - 题目排名分别显示个人排名和队伍排名，个人排名根据学生提交的子题目分数得分率进行排名，队伍排名根据队伍内所有学生提交的分数总和得分率进行排名。
